Here's a simple Python program that demonstrates recursion by calculating the factorial of a number:

Python Program for Factorial Using Recursion

python
def factorial(n):
    if n == 0 or n == 1:
        return 1  # Base case
    else:
        return n * factorial(n - 1)  # Recursive case

# Taking user input
num = int(input("Enter a number: "))

if num < 0:
    print("Factorial is not defined for negative numbers.")
else:
    print(f"Factorial of {num} is {factorial(num)}")

How It Works:

  1. If n is 0 or 1, return 1 (base case).
  2. Otherwise, return n * factorial(n - 1), which calls the function recursively.
